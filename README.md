# ✅ To-Do App (React.js)

The **To-Do App** is a simple, responsive, and interactive task management application built using **React.js**. It helps users manage daily tasks efficiently by allowing them to add, delete, and mark tasks as completed.

This project focuses on learning React fundamentals like state management, event handling, and dynamic UI updates.

---

## 🚀 Features

- 📝 Add new tasks  
- ✔️ Mark tasks as completed  
- ❌ Delete tasks easily  
- ⚡ Real-time UI updates  
- 📱 Fully responsive design  
- 🎨 Clean and minimal interface  

---

## 🧰 Tech Stack

- React.js (Create React App)  
- JavaScript (ES6+)  
- CSS3 / Tailwind CSS  
- React Hooks (useState)  

---

## 🎯 Purpose

This project was built to practice **React basics**, especially:

- State management using `useState`  
- Handling user input  
- Conditional rendering  
- Working with lists and events  
- Building reusable components
